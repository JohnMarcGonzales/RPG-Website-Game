Command List

  • go [direction] - Move (e.g., go north)
  • look - Look around
  • take [item/number] - Take item
  • attack - Attack enemy
  • heal - Use potion
  • item - Show inventory
  • equip [item/number] - Equip item
  • escape - Try to escape
  • talk - Talk to NPC
  • rest - Rest at inn
  • history [n] - Show last n history entries
